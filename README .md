### 🚀 Featured Projects

**[🛰️ Satellite Scene Classification](https://github.com/fidalgovind/satellite-scene-classification)**
Multi-class land-use classification on the EuroSAT (Sentinel-2) dataset using a fine-tuned ResNet-18.
- **92.89% accuracy**, 0.93 macro F1, across 10 land-use classes
- Engineered NDVI (vegetation index) as an additional feature from spectral bands
- Added Grad-CAM explainability to visualize model attention
- Deployed as a live [Streamlit app](https://satellite-scene-classification-4remxga6jkwsfybr79ivjq.streamlit.app) for interactive predictions
- `Python` `PyTorch` `ResNet-18` `Grad-CAM` `Streamlit`

**[📧 Email Spam Classification](https://github.com/fidalgovind/Email-Spam-Classification)**
NLP-based spam/ham classifier trained on the SpamAssassin dataset, comparing 4 ML algorithms.
- Built a full text pipeline: tokenization, stopword removal, stemming, TF-IDF vectorization
- Benchmarked Naive Bayes, Logistic Regression, SVM, and Random Forest — **SVM performed best**
- Deployed the winning model as a live [Streamlit app](https://email-spam-classification-erd32gzrsof4qto2zxkfz8.streamlit.app/) for real-time predictions
- `Python` `NLTK` `Scikit-learn` `TF-IDF` `Streamlit`

> 💡 *More projects in progress — including a CUDA/OpenMP/MPI parallel computing study and a BigQuery + Looker Studio cloud analytics case study.*
